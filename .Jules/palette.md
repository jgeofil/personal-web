## 2024-05-24 - Accessibility of Icon-only Links and Decorative Images
**Learning:** Icon-only links (like SocialGrid) must have a dynamic `aria-label` attribute on the `<a>` element to be accessible to screen readers, and the accompanying `<img>` should be hidden from assistive technologies using `alt=""` and `aria-hidden="true"`. Purely decorative images (like LinkGrid) adjacent to text links must also be explicitly hidden using `alt=""` and `aria-hidden="true"`.
**Action:** Always ensure any link composed solely of an image has an explicit text alternative via `aria-label`, and always hide decorative images (whether they are the sole content of a link or placed alongside text) using `alt=""` and `aria-hidden="true"` to prevent redundant or confusing screen reader announcements.
